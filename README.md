This branch is configured for PostgreSQL:

- ICAT database:

    - PostgreSQL database: `ICAT`
    - PostgreSQL user:     `icat`
    - PostgreSQL pass:     `icat`

- iRODS:

    - iRODS user:          `irods`
    - iRODS pass:          `irods`

A clean ICAT database dump is provided in the file icat_dump.sql
